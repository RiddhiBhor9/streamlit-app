# 📊 Streamlit Data Dashboard

An interactive data visualization web app built using **Streamlit** and Python. This project allows users to explore, analyze, and visualize datasets through an easy-to-use interface.

---

## 🚀 Features

* 📈 Interactive charts & graphs
* 🔍 Data filtering and exploration
* 📊 Real-time visualization
* 💻 Simple and clean UI

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Plotly / Matplotlib

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/RiddhiBhor9/streamlit-app.git
cd streamlit-app
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run app.py
```
